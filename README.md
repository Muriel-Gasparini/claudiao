# claudiao

TUI installer for a Spec Driven Development (SDD) framework for
[Claude Code](https://claude.com/claude-code).

Install curated `rules/`, `commands/`, `agents/`, `output-styles/`, and
spec templates into `~/.claude/` without touching your personal data
(memory, conversations, credentials).

## Status

Work in progress. MVP under active development.
